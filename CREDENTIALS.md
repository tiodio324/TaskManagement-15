# Учетные данные для входа в систему

## Роли пользователей

Система поддерживает следующие роли:

- **user** (Гость) - роль по умолчанию, не требует авторизации
- **developer** (Разработчик) - требуется авторизация с паролем
- **admin** (Администратор) - требуется авторизация с паролем

## Пароли для входа

| Роль | Пароль |
|------|--------|
| Разработчик (developer) | `developer2026-tasks` |
| Администратор (admin) | `admin2026-tasks` |
